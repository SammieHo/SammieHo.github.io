client 客户区坐标位置

顾名思义，相对于 浏览器内容区域 视口 而言的clientX，clientY （不包含页面滚动的距离）

screen 屏幕坐标位置

相对于 整个显示器的屏幕而言的 screenX，screenY

page，offset 页面坐标位置

相对于整个网页，是算上滚动条的。

例如：
网页的 width：980px; height: 3000px;
那么 page 和 offset 的最大值就是 980 和 3000。

而 page 和 offset 的区别就是：IE8以下只支持offset。
