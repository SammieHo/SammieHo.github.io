    截断一个字符串

    规则：

    如果字符串的长度比指定的参数num长，则把多余的部分用…来表示。

    切记，插入到字符串尾部的三个点号也会计入字符串的长度。

    但是，如果指定的参数num小于或等于3，则添加的三个点号不会计入字符串的长度。

    思路：

    1. 对比字符串strLen与参数num；
    2. strLen > num ,截断str范围(0,num-3]，加”…”
    3. strLen <= 3,不截断，直接加“…”
