实践思考

我的Demo：flexbox练手——音乐界面布局Demo

我想说的

Set a certain parent element to be a flex-container by using display: flex || inline-flex.
不知为啥，全天下来，这句话一直在脑子里面啊飘啊飘，大概是打过鸡血的缘故~~

这个Demo里，一直嵌套，借文章一句“炫技”，囫囵吞枣，都用flexbox 布局。

然而，我发现用flex 来布局 导航、卡片 是非常顺手的！！

远离 float，div，table 这些的，耳根清净大概这个意思！！

当然，是在不考虑兼容的情况下，微笑:)

像这种，分分钟的事情~

flex-container 中用得最多的属性（父元素）：

  1. 设置排列顺序，非常常用！！
    flex-direction: row（默认）|| column ;

  2. 是否换行
    flex-wrap: nowrap(默认不换行) || wrap ;

  3. 主轴方向 Main Asix（水平的）的对齐
    justify-content: center || space-between(两端对齐)
    注意：若设置了margin的auto，该值失效。

flex-items 常用的属性：

  1. 灵活性（这里还是有点不明白，之前补充）
    flex-glow 是否填充可用空间，默认 0；
    flex-shrink 是否收缩，默认 1；
    flex-basis 指定Flex项目的初始大小，默认auto；

  2. 缩写：
    flex：flex-glow flex-shrink flex-basis;
    flex: 0 1 auto; // 默认行为
    flex: 0 0 auto; // 相当于 flex: none;
    flex: 1 1 auto; // 相当于 flex: auto;
